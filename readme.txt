Add WayPay Gateway to WooCommerce

== Description ==

ONLY AVAILABLE FOR ‘BRL’ CURRENCY

SOMENTE DISPONÍVEL PARA TRANSAÇÕES EM MOEDA BRASILEIRA (BRL).

Adicione o WayPay como método de pagamento em sua loja WooCommerce.

- Venda com Cartão de Crédito ou Boleto!

Este plugin adiciona o gateway [WayPay](http://www.waypay.com.br/) à sua loja WooCommerce.

= Compatibilidade =

- [WooCommerce] (http://wordpress.org/plugins/woocommerce/) 3.x ou posterior.
- [WooCommerce Extra Checkout Fields for Brazil] (http://wordpress.org/plugins//woocommerce-extra-checkout-fields-for-brazil/) 3.6.1 ou posterior.
- E um certificado SSL e a opção "Forçar finalização segura" ativada.

== Installation ==

- Envie os arquivos do plugin para a sua pasta de plugins ou instale usando o instalador interno do WordPress "Adicionar Novo Plug-in".
- Ativar o plugin.
- Uma vez instalado o plugin você precisa ir em "WooCommerce"> "Configurações"> "Checkout"> "WayPay - Ticket" ou "WayPay - Cartão de Crédito" e marcar a habilitação do gateway e preencher as opções.
